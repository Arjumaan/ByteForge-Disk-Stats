## â“ Frequently Asked Questions

### Why does it need administrator rights?
Some features require elevated permissions:
- Reading CPU temperature
- Accessing system files
- Managing processes
- Cleaning system folders

### Is it safe?
Yes! The app only monitors and manages your system. It doesn't:
- Connect to the internet
- Send your data anywhere
- Install malware or adware
- Modify critical system files

### Can I run it without admin rights?
Yes, but some features will be limited:
- CPU temperature will show "N/A"
- Some cleanup operations won't work
- Registry access will be restricted

### Why does my antivirus flag it?
Some antivirus software shows false positives for:
- Unsigned executables
- Apps requesting admin rights
- System monitoring tools

**Solution:** Add an exception in your antivirus settings.
